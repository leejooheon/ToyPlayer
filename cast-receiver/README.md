## 1. Prerequisites
- Node.js and npm installed
- Firebase CLI installed
- Registering the Receiver App: https://cast.google.com/publish/#/overview

```bash
npm install -g firebase-tools
```

## 2. Login to Firebase
```bash
firebase login
```

## 3. Initialize Firebase
```bash
firebase init
```

## 4. Deploy to Firebase Hosting
```bash
firebase deploy
```
